## server action practice

server action 을 이용한 api 요청 연습입니다.

## Tech Stack

- Next.js
- TypeScript
- Framer
- Tailwind CSS

## Quick Start

### Clone
```
git clone https://github.com/HMRyu/server-action-practice.git
```

### Installation
```
npm install
```

### Run
```
npm run dev
```
